# TailPlanner – MVP Execution Checklist (Strict)

## Purpose

- First usable version  
- Real daily usage possible  
- Data correctness over appearance  

---

## MVP Must Include

- List creation & items  
- Todo (Today / Overdue / Undated)  
- Todo → Timeline  
- Focus (single task only)  
- Focus → Timeline  
- Timeline view & edit  
- Plan basic page (read-only aggregation)  

---

## MVP Explicitly Excludes

- ❌ Multitask focus  
- ❌ Today page  
- ❌ Statistics  
- ❌ UI polish  

---

## MVP Completion Definition

> **MVP = 55% of the full system**  
> All core data flows are validated, appearance is intentionally minimal.
